# Project screenshots

Drop a real screenshot for each project here and reference it from the
`image` field in `src/components/sections/Projects.tsx`.

Recommended: 1600×1000 px (16:10), `.png` or `.webp`, project chrome cropped out.

| Project          | Suggested filename     |
| ---------------- | ---------------------- |
| Evano Studio     | `evano-studio.png`     |
| CrewX            | `crewx.png`            |
| EvanoTask        | `evano-task.png`       |
| Evano Development| `evano-dev.png`        |
| El Inka Envíos   | `inka-envios.png`      |

When `image` is set on a project, the card renders that screenshot inside the
browser frame. When it is empty, a branded placeholder mockup is shown instead.
